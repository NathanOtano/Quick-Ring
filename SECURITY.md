# Sécurité

Signalez un problème via les issues GitHub de ce dépôt, sans inclure de données personnelles ni de contenu de calendrier.

Pour un signalement utile, indiquez :

- la version concernée ;
- le système utilisé ;
- les étapes minimales de reproduction ;
- le résultat attendu et observé.

Ne joignez pas de fichiers de configuration Google, de journaux privés, de captures contenant des événements personnels ou de données de calendrier.
